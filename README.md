
# EVENT-HUB

EVENT-HUB is a simple student-friendly web application that showcases upcoming campus events, workshops, socials, and career opportunities.  
It allows students to browse events, search by keywords, and RSVP directly via email.

---

##  Features
- **Event Listings**: Displays multiple event cards with title, date, and description.
- **Search Bar**: Quickly filter events by title, tag, or location.
- **Category Filters**: Buttons to view events by type (Workshops, Socials, Career).
- **Responsive Layout**: Works across desktop and mobile devices.

---

##  Technologies Used
- **HTML5**: Semantic structure for event cards, search, and filters.
- **CSS3**: Custom styling with gradients, cards, buttons, and responsive design.
  - CSS Variables (`:root`) for theme colors and reusability.
  - Flexbox and Grid for layout.
  - Hover effects for interactive cards and buttons.
- **Git**: Version control for tracking changes and collaboration.
